# 🚀 QUICKSTART - Empire Coach

**Installation en 3 minutes sur iOS**

## Étape 1 : Lance le serveur

```bash
cd /data/.openclaw/workspace/projects/empire-coach
python3 serve.py
```

Tu verras :
```
🎮 EMPIRE COACH - Serveur Local
✅ Serveur démarré sur le port 8000
📱 Sur ton iPhone, ouvre Safari et va sur :
   http://192.168.1.X:8000
```

## Étape 2 : Ouvre sur ton iPhone

1. Ouvre **Safari** (important, pas Chrome)
2. Entre l'URL affichée : `http://192.168.1.X:8000`
3. L'app se charge

## Étape 3 : Installe sur l'écran d'accueil

1. Appuie sur le bouton **Partager** (carré avec flèche vers le haut)
2. Scroll down
3. Choisis **"Sur l'écran d'accueil"**
4. Nomme l'app **"Empire Coach"**
5. Appuie sur **Ajouter**

## Étape 4 : Lance l'app

1. L'icône apparaît sur ton écran d'accueil
2. Appuie dessus comme une app native
3. Entre ta clé API Anthropic quand demandé
4. **C'est parti ! 🎮**

---

## 🔑 Obtenir une clé API (gratuite)

1. Va sur https://console.anthropic.com
2. Crée un compte (email + mot de passe)
3. Tu reçois **$5 gratuits**
4. Va dans **API Keys**
5. Clique **Create Key**
6. Copie la clé (elle commence par `sk-ant-...`)
7. Garde-la précieusement (tu en auras besoin à chaque session)

**Coût** : ~$0.01-0.05 par session. Avec $5, tu as ~100-500 sessions.

## 🎮 Utilisation

### Interface

- **Splash** → Appuie sur "COMMENCER"
- **Évaluation initiale** → Réponds honnêtement aux questions
- **Map** → Navigue entre les 12 piliers
- **Leçon** → Chat avec le coach, gagne de l'XP
- **Stats** → Panneau de progression détaillé

### Gamification

- **XP** : +50 base, +100 bonne réponse, +200 parfait
- **Rangs** : Apprenti → Marchand → Stratège → Magnat → Empereur
- **Gemmes** : 1 💎 par réponse parfaite (≥9/10)
- **Streak** : 🔥 bonus si tu reviens chaque jour

## 🐛 Problèmes ?

**L'app ne s'installe pas** :
- Utilise Safari (pas Chrome)
- Vérifie que tu es sur le bon réseau Wi-Fi
- Recharge la page

**L'API ne répond pas** :
- Vérifie ta clé API
- Regarde tes crédits sur console.anthropic.com
- Vérifie ta connexion internet

**Autre problème** :
- Lis le README.md complet
- Vérifie la console navigateur (Safari > Développer > Console)
